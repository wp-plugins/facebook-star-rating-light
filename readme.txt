Find the Facebook Rating admin page in the settings sub menu.2. Register your Facebook page name.3. Register your Facebook page url.4. Decide if you want to have the rating text to be displayed next to the stars.5. Define the styling of the rating text in the given textarea using css styling rules or accept the default.6. Now save your settings. The Facebook Page rating should appear after a couple of seconds.7. Use [fb_stars] as shortcode inside your posts, pages, or widgets.== Frequently Asked Questions === How can I establish a automatic update? =*Facebook Star Rating Light* is limited and does not support automatic exchangerate updates.Please find it's siblings with additional options at [intelligent-it.asia](http://intelligent-it.asia).FACEBOOK STAR RATING BRONCE already has automatic update functionality.= How can I upload my custom star images? =*Facebook Star Rating Light* does not support custom star images. Go for "Facebook Star Rating Gold" available at [intelligent-it.asia](http://intelligent-it.asia).= How often should I update the rating? =With *Facebook Star Rating Light* it depends on you, whenever you visit the settings page the rating will be updated.But don't forget to hit the "save changes" button.== Screenshots ==1.
